🐛 Troubleshooting
Common Issues:

CSS not loading?

Check that styles.css is in the same folder as index.html
Ensure file name is exactly styles.css (not style.css)


JavaScript not working?

Check that script.js is in the same folder
Open browser console (F12) to check for errors


Links not working?

Update placeholder URLs with your actual project links
Ensure external links include https://


Mobile menu not working?

Clear browser cache and refresh
Check that JavaScript is enabled
